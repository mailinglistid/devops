git init : initialize local Git Repository
git config --global user.name "Raja" : Tell Github who you are by configuring author name.
git config --global user.email raja1@gmail.com : Tell Github who you are by configuring author email.
git add <filename>/ git add . : add one or more files to stagging area.
git commit -m "commit message" : Commit changes to head but not to the remote Repository.
git status: check the status of your current repository and list the files you have changes.
git log : provide a list of all the commit made on your branch.
git log --oneline : provide a list of all the commit made on your branch in one line.
git diff : view the changes you have made to the file.
git push origin <branch name> : push the code to the remote repository so that other can see.
git clone : create a git repository copy from a remote source.
git remote add origin <server> : Connect your local repository to the remote server and add the server to be able to push to it.
git branch <branch name>: Create a new branch.
git checkout <branch name> : Switch from one branch to another.
git merge <branch name> : merge a branch into the active branch.
git reset <commit id> : reset the change in the local repo.
git revert <commit id> : reset the change in the remote repo.
git show <commit id> : show the change done in the commit.
